# sudoku_all

Python Flask starter workspace.

## Funktioner

- Knapp pa startsidan till Sudoku-byggaren.
- Tomt 9x9-rutnat for att fylla i egna siffror.
- Spara grund-spel med namn och svarighetsgrad (latt, medel, svar, expert).
- Egen sida for pågående spel, separat fran grund-spel.

## Kom igang

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python run.py
```
